# 災害AR PWA（AI自動記録版）

GitHub Pages にそのまま置ける、災害AR試作アプリです。

## 追加した機能
- 実地図 + 避難所ルート表示
- AR画面で次の進行方向を大きな矢印で表示
- ブラウザ軽量AI（COCO-SSD）による物体検出
- 検出時の自動シャッター
- 数秒間隔の自動シャッター
- 自動保存された画像付き投稿
- 投稿のあとから修正
- JSON / GeoJSON エクスポート

## 重要
この版のAIは **GitHub Pages 上で動かしやすい軽量ブラウザ実装** です。
本格的な Grounding DINO や被害判定モデルを高精度で回す構成ではありません。

### いま出来ること
- 人・車・トラックなどの一般物体検出
- 検出トリガーで画像を自動保存
- 画像を残して手動修正

### まだ弱いところ
- 倒壊・浸水・火災などの被害種別を高精度に自動判定する機能
- DINO/GLIP 系の重いゼロショット検出を iPhone のブラウザだけで安定動作させること

## GitHub Pages 公開
1. このフォルダの中身をリポジトリ直下へ置く
2. Settings → Pages
3. `Deploy from a branch` なら `main / (root)` を選ぶ
4. もしくは同梱の `.github/workflows/pages.yml` を使って Actions で公開

## 使い方
1. 地図で避難所を選び、ルートを出す
2. ARで「カメラ開始」
3. 「AI読込」→「AI開始」
4. 自動保存モードを選ぶ
5. 投稿一覧で AI 自動保存画像を確認し、必要なら「修正」

## 注意
- HTTPS で配信してください
- 初回は位置情報、カメラ、向きの許可が必要です
- AIモデル読込時は通信が必要です
